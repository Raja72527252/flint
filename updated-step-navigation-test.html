📖 Layout Logic Explained

Completed Steps
✓
Checkmark Only

जो steps पहले complete हो चुके हैं, वे केवल green checkmark (✓) show करते हैं। Number hide हो जाता है क्योंकि completion ज्यादा important है।

  • • Color: Green (#36D399)
  • • Symbol: ✓ checkmark
  • • Number: Hidden
Current Step
3
Number Visible

Current active step में number clearly visible है blue background के साथ। यह user को बताता है कि वे currently कौन से step पर हैं।

  • • Color: Blue (#6E0048)
  • • Symbol: Step number
  • • Highlight: Enlarged & highlighted
Pending Steps
5
Number Visible

Future steps में numbers visible हैं gray outline के साथ। Users को पता चलता है कि आगे कौन से steps आने वाले हैं।

  • • Color: Gray outline
  • • Symbol: Step number
  • • Status: Clickable for navigation
Connectors
vs

Steps के बीच connecting lines भी status show करती हैं। Completed connections green हैं, pending connections gray हैं।

  • • Completed: Green line
  • • Pending: Gray line
  • • Flow: Left to right progression
